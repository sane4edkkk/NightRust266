Описание

IQKITS - это мощный плагин для управления китами в игре. Позволяет создавать, редактировать и выдавать наборы предметов игрокам с различными настройками задержки и прав доступа.

Команды

/kit create [название]
Создать новый кит из предметов в вашем инвентаре
/kit delete [название]
Удалить существующий кит
/kit give [игрок] [название]
Выдать кит указанному игроку
/kit list
Показать список доступных китов
